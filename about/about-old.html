{{ site.title }} Pure

{{ site.title }} Pure is a true, direct fork of the Unity 7 desktop environment. Bringing the latest innovations from {{ site.title }} Core to the well-trusted and venerable Unity 7 desktop, {{ site.title }} Pure aims to be a replacement to the Unity 7 desktop for those who prefer X11 and Compiz as their workflow.
