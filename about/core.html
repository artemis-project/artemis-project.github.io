Unity, in Plasma

{{ site.title }} Core is the new, innovative desktop environment that continues the legacy of Unity 7. Built upon the work of Plasma, Core brings new features and support for modern technologies like Wayland and universal search.

More than Look&Feel

{{ site.title }} Core is available as a Look&Feel package, giving existing Plasma users an easy transition to the {{ site.title }} desktop. However, we do have plans to make an installable package via Snappy and/or Flatpak that bundles Plasma and {{ site.title }} together, with {{ site.title }} set as the default Look&Feel package for Plasma.
