The next generation of Unity 7

The Enjade Project is a continuation of the Unity 7 desktop millions of Ubuntu users are already familiar with, originally created by Canonical.

Enjade Core

Enjade Core is the Plasma-based, modern Unity 7 desktop environment. Including new innovations and support for Wayland, Enjade Core provides the flexibility of the Plasma environment while bringing the familiarity of Unity to users.

Learn more ›

Enjade Pure

Enjade Pure is a true, direct fork of the Unity 7 desktop environment. Bringing the latest innovations from Enjade Core to the well-trusted and venerable Unity 7 desktop, Enjade Pure aims to be a replacement to the Unity 7 desktop for those who prefer X11 and Compiz as their workflow.

enjade /ɛn - 'jeɪd/*

The word enjade came about as a result of Acquainted Safari's potentialℨ partnership with Jaeden Gurrero. Essentially, the pronunciation of the syllables have been swapped, creating a unique word that represents the spirit of continuing the venerable Unity desktop.

While her personality does match up with the spirit and community formed to continue Unity, she is not affiliated with this project by any means.

* This pronunciation key uses the Oxford dictionary syntax.

✝ Enjade is not to be confused with the Malay word "enjade", meaning "get".

ℨ The status remains as 'potential' at the time of writing this page.
