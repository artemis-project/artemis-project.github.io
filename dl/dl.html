Enjade Core

Enjade Core is the traditional Plasma-based desktop that continues Unity 7 through means of Plasmoids. Core provides support for Wayland and often get new features before Enjade Pure.

The Ubuntu distro with Enjade Core uses the long-term support releases (LTS). Learn more ›

Enjade Pure

Enjade Pure is the Unity 7 desktop fork that provides the features of Enjade Core. Pure provides support for X11 and Compiz, usually getting features after Core implementation.

The Ubuntu distro with Enjade Pure uses the regular, time-based releases. Learn more ›
