Meet the {{ site.title }} family.

Whether you prefer using a Plasma-based environment with new innovations or a classic environment with feature updates and solid stability, the {{ site.title }} family of environments and projects has you covered.

Core

Looking for a Plasma-based environment that brings a wonderful Unity experience while getting new features continuously? Core brings new features constantly and supports new technologies in the modern Linux space like support for Wayland. With a plethora of plasmoids and a comprehensive Look&Feel package, Core brings the best experience for those who want to be on the bleeding edge of technology.

Pure

Want a stable, reliable, and venerable desktop environment while still getting new features? Pure is a direct fork of the Unity desktop, based off of X11 and Compiz technologies. While the cycle for feature development is much slower, it provides a solid experience, ready for any user who wants something that "just works".
